## All packages

```bash
# pacman

$ sudo pacman -Sy fish jq aria2 neovim micro lsd xfce4-screenshooter

# yay

$ yay -Sy visual-studio-code-bin librewolf-bin ytop-bin nerdfetch ulauncher

# orphaned pkgs

$ yay --clean

# clear pacman cache of pkgs that have been removed

$ sudo pacman -Sc
```
