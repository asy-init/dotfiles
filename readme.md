## Dotfiles 
###### ( work in progress do not use )

`Installation`
```
$ git clone https://github.com/asy-init/dotfiles

$ cd dotfiles

$ chmod u+x ./scripts

$ sh ./scripts/launch.sh
```
or 
```
git clone https://github.com/asy-init/dotfiles;cd dotfiles;chdmod u+x ./scripts;sh ./scripts/launch.sh
```
