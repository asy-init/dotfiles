## Setup

1. Open Xfce Terminal Emulator 

```bash
cd ~/.local/share/konsole
rm -r *

cd ~/.config/
rm konsolerc
```

Move all items in:



```
#dotfiles/konsole/.config   into /home/asy/.local/share/konsole/
```


```
#dotfiles/konsole/konsolerc into /home/asy/.config/
```
